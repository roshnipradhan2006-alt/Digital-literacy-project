Task 1 – Digital Literacy Infographic README

For the purpose of completing task 1, it is required to prepare an infographic, and the infographic must be related to the concept of digital literacy. For this reason, an infographic has been prepared using an online tool, and in the infographic, information related to digital literacy, such as internet etiquette, tools for students, and professionalism, has been included. The main aim of this infographic is to spread awareness among students regarding digital literacy.
